**Architecture:** Replace the `int cents` / `int contracts` money-unit invariant with `int bps` (`1¢ = 100 bps`) and `int fp100` (`1 contract = 100 fp100`). Decimal parser at the wire boundary fails closed on sub-bps/sub-fp100 precision. Versioned (v2) ledger envelope embeds a `legacy_v1_snapshot` blob until reconciled against authoritative Kalshi fills. Risk-increasing money actions gated until the ledger is ready; cancel path is carve-out with F33 resync. AST test locks the invariant after migration.

**Tech stack:** Python 3.12+, async-first, Pydantic v2, Decimal at wire boundary, structlog, pytest. Single-event-loop asyncio (v11 simplification — every ledger mutator is a sync `def`).

## Goal

Eliminate the two architectural failures documented in the spec:

1. Sub-cent-tick markets (e.g. DJT at `3.8¢/96.1¢`) silently dropped at `src/talos/scanner.py:182` because `dollars_to_cents` rounds sub-cent prices to an integer cent.
2. Fractional maker fills (e.g. MARJ `1.89` contracts at `48.88¢`, observed 2026-04-21) silently truncated at `src/talos/models/_converters.py:39` (`int(float("1.89"))` → `1`), inflating cost basis and corrupting inventory.

Both stem from the same integer-cents / integer-contracts invariant embedded across 24 files. Phase 1+2 replaces that invariant with `int bps` for money (`1 bps = $0.0001`, `1¢ = 100 bps`) and `int fp100` for counts (`1 contract = 100 fp100`), adds fail-closed Decimal parsers at the wire boundary, persists a v2 ledger envelope with an embedded legacy blob until reconciled, gates risk-increasing actions on a fills-based authoritative reconcile, and locks the invariant with an AST regression test.

Talos must continue to submit whole-contract orders only (fractional submission is out-of-scope); fractional arrivals via maker fills must be represented correctly once received. After this lands, admission drops the Phase-0 fractional/sub-cent guards.

## Constraints and assumptions

- **Python 3.12+, async-first, Pydantic v2, Decimal at wire boundary.** `float()` on `_dollars` or `_fp` wire strings is banned outside `units.py`.
- **Single-event-loop asyncio (v11 simplification).** Every ledger mutator is a sync `def`. No `asyncio.Lock`, no global persistence lock. The AST/unit-discipline tests actively reject any re-introduction of `PositionLedger._mutation_lock` or `Engine._persistence_lock`.
- **Kalshi is the source of truth** (CLAUDE.md cardinal rule, Principle 15). Any ledger field sourced from persistence is treated as stale until an authoritative Kalshi source re-confirms it this session.
- **Wire-format contract is conditional, settled:** `bps_to_dollars_str` emits 2-decimal strings when `bps % ONE_CENT_BPS == 0` and 4-decimal strings otherwise. Always-4-decimal was considered and rejected (round 13 of design).

   ```python
   # tools/migrate_v2_to_v1.py
   """Extract v1-compatible ledger payload from a v2 games_full.json.

   Lossy: fractional contract counts (fp100 values not divisible by 100)
   cannot be represented in v1 and are surfaced as warnings. The operator
   must resolve those positions on Kalshi before reverting to a legacy
   binary, or accept the integer-contract truncation.
   """
   from __future__ import annotations
   import argparse
   import json
   import sys
   from pathlib import Path

   V2_TO_V1_KEY_MAP = {
       "filled_count_fp100_a":   ("filled_a",          100),   # fp100 → contracts
       "filled_total_cost_bps_a":("cost_a",            100),   # bps → cents
       "filled_fees_bps_a":      ("fees_a",            100),
       "filled_count_fp100_b":   ("filled_b",          100),
       "filled_total_cost_bps_b":("cost_b",            100),
       "filled_fees_bps_b":      ("fees_b",            100),
       "closed_count_fp100_a":   ("closed_count_a",    100),
       "closed_total_cost_bps_a":("closed_total_cost_a",100),
       "closed_fees_bps_a":      ("closed_fees_a",     100),
       "closed_count_fp100_b":   ("closed_count_b",    100),
       "closed_total_cost_bps_b":("closed_total_cost_b",100),
       "closed_fees_bps_b":      ("closed_fees_b",     100),
       "resting_count_fp100_a":  ("resting_count_a",   100),
       "resting_price_bps_a":    ("resting_price_a",   100),
       "resting_count_fp100_b":  ("resting_count_b",   100),
       "resting_price_bps_b":    ("resting_price_b",   100),
       # pass-through (no scaling)
       "resting_id_a":           ("resting_id_a",      1),
       "resting_id_b":           ("resting_id_b",      1),
   }

   def _convert_ledger(v2_ledger: dict, pair_name: str) -> tuple[dict, list[str]]:
       """Return (v1_ledger, warnings). Warnings list is empty on lossless conversion."""
       warnings: list[str] = []
       v1: dict = {}
       for v2_key, (v1_key, scale) in V2_TO_V1_KEY_MAP.items():
           val = v2_ledger.get(v2_key)
           if val is None or scale == 1:
               v1[v1_key] = val
               continue
           if val % scale != 0:
               warnings.append(
                   f"{pair_name}: {v2_key}={val} is not divisible by {scale}; "
                   f"lossy truncation to v1 {v1_key}={val // scale}"
               )
           v1[v1_key] = val // scale
       return v1, warnings

   def main() -> int:
       ap = argparse.ArgumentParser()
       ap.add_argument("v2_path", type=Path)
       ap.add_argument("--out", type=Path, required=True)
       args = ap.parse_args()

       payload = json.loads(args.v2_path.read_text())
       all_warnings: list[str] = []
       for pair_name, entry in payload.get("games", {}).items():
           ledger_blob = entry.get("ledger")
           if not isinstance(ledger_blob, dict):
               continue
           if ledger_blob.get("schema_version") != 2:
               # Already v1 shape or unknown — leave unchanged.
               continue
           # Prefer the embedded legacy_v1_snapshot when it exists (lossless).
           if "legacy_v1_snapshot" in ledger_blob:
               entry["ledger"] = ledger_blob["legacy_v1_snapshot"]
               continue
           # Otherwise reverse the v2 payload.
           v1, warnings = _convert_ledger(ledger_blob["ledger"], pair_name)
           all_warnings.extend(warnings)
           entry["ledger"] = v1

       args.out.write_text(json.dumps(payload, indent=2) + "\n")
       if all_warnings:
           print("LOSSY CONVERSION WARNINGS:", file=sys.stderr)
           for w in all_warnings:
               print(f"  - {w}", file=sys.stderr)
           return 2   # nonzero exit on lossy conversion; operator must confirm
       return 0

   if __name__ == "__main__":
       sys.exit(main())
   ```

```python
def seed_from_saved(self, data: dict | None) -> None:
    if data is None:
        return                                         # no-op, all flags False

    schema = data.get("schema_version")
    if schema is None or schema < 2:
        # v1 load path. data IS the legacy ledger dict (cents/contracts).
        return self._seed_from_v1(data)
    if schema == 2:
        return self._seed_from_v2(data)
    raise ValueError(f"unsupported ledger schema_version: {schema}")


def _seed_from_v1(self, v1: dict) -> None:
    """v1 keys → v2 fields, ×100 per the spec key map."""
    v2_ledger: dict = {}
    for v1_key, v2_key, scale in _V1_TO_V2_KEY_MAP:    # module-level constant
        val = v1.get(v1_key)
        if val is None or scale == 1:
            v2_ledger[v2_key] = val
        else:
            v2_ledger[v2_key] = int(val) * scale
    self._apply_loaded_state(v2_ledger)
    stale_fills, stale_resting = _staleness_from_loaded(v2_ledger)
    self.stale_fills_unconfirmed = stale_fills
    self.stale_resting_unconfirmed = stale_resting
    # F22: legacy_migration_pending only set when v1 had nonzero safety state.
    if stale_fills:
        self.legacy_migration_pending = True
        self._legacy_v1_snapshot = dict(v1)            # retain verbatim


def _seed_from_v2(self, envelope: dict) -> None:
    ledger_dict = envelope["ledger"]
    self._apply_loaded_state(ledger_dict)
    stale_fills, stale_resting = _staleness_from_loaded(ledger_dict)
    self.stale_fills_unconfirmed = stale_fills
    self.stale_resting_unconfirmed = stale_resting
    self.legacy_migration_pending = bool(envelope.get("legacy_migration_pending"))
    if self.legacy_migration_pending:
        self._legacy_v1_snapshot = envelope.get("legacy_v1_snapshot")
```

The `_V1_TO_V2_KEY_MAP` is the spec Section 7 table (lines 391–410) encoded as a module-level tuple-of-tuples; pass-through entries (`resting_id_a`, `resting_id_b`) use `scale = 1`.

The `_staleness_from_loaded` helper returns `(stale_fills, stale_resting)` based on whether any historical / resting field is nonzero / non-null:

```python
def _staleness_from_loaded(d: dict) -> tuple[bool, bool]:
    fills_relevant = (
        d["filled_count_fp100_a"], d["filled_count_fp100_b"],
        d["filled_total_cost_bps_a"], d["filled_total_cost_bps_b"],
        d["filled_fees_bps_a"], d["filled_fees_bps_b"],
        d["closed_count_fp100_a"], d["closed_count_fp100_b"],
        d["closed_total_cost_bps_a"], d["closed_total_cost_bps_b"],
        d["closed_fees_bps_a"], d["closed_fees_bps_b"],
    )
    resting_relevant = (
        d["resting_count_fp100_a"], d["resting_count_fp100_b"],
        d["resting_id_a"], d["resting_id_b"],
    )
    stale_fills = any(v != 0 for v in fills_relevant)
    stale_resting = any(v not in (0, None) for v in resting_relevant)
    return stale_fills, stale_resting
```

```python
# src/talos/engine.py

async def create_order_with_gate(self, pair, **kw) -> Order | None:
    """Gated entry point for every money-increasing create. Blocks until
    pair.ledger.ready() or until STARTUP_SYNC_TIMEOUT_S expires, whichever
    is first. On block/timeout, surfaces a notification and returns None
    (callers must handle None as "did not submit"). F31 carve-out does not
    apply here — create always increases risk; cancel is the only op that
    bypasses the gate."""
    if not await self._wait_for_ledger_ready(pair, op_name="create_order"):
        return None
    return await self._rest.create_order(**kw)

async def amend_order_with_gate(self, pair, **kw) -> tuple[Order, Order] | None:
    if not await self._wait_for_ledger_ready(pair, op_name="amend_order"):
        return None
    return await self._rest.amend_order(**kw)

async def decrease_order_with_gate(self, pair, **kw) -> Order | None:
    if not await self._wait_for_ledger_ready(pair, op_name="decrease_order"):
        return None
    return await self._rest.decrease_order(**kw)
```

```python
    try:
        save_games([p.event_ticker for p in game_mgr.active_games])
```

```python
# tests/test_unit_discipline.py
import ast
import pathlib

MONEY_IDENT_PATTERNS = ("price", "cost", "bps", "fp100", "cents", "fees", "edge")
BANNED_LITERALS = {100, 10_000}
BANNED_FORMAT_SPECS = {".2f", ".4f"}

ALLOWED_LITERALS = {
    # (path, lineno, reason)
    # time conversions
    ("src/talos/ui/tree_screen.py", None, "elapsed_ms time formatting, not money"),
    # cpm ratio formatting
    ("src/talos/cpm.py", 37, "ratio :.2f, not price"),
    # ... add entries as surfaced
}

def _ident_matches_money(name: str) -> bool:
    lower = name.lower()
    return any(p in lower for p in MONEY_IDENT_PATTERNS)

def test_no_raw_money_unit_arithmetic():
    violations = []
    for py in pathlib.Path("src/talos").rglob("*.py"):
        if py.name == "units.py" or py.name == "_converters.py":
            continue   # units.py allowed; _converters.py has deprecated shim

        tree = ast.parse(py.read_text())
        for node in ast.walk(tree):
            # Check BinOp with a literal 100 or 10_000 alongside a money ident
            if isinstance(node, ast.BinOp):
                for side in (node.left, node.right):
                    if isinstance(side, ast.Constant) and side.value in BANNED_LITERALS:
                        other = node.right if side is node.left else node.left
                        if _is_money_expr(other):
                            key = (str(py), side.lineno, "money arith with literal 100/10_000")
                            if key not in ALLOWED_LITERALS:
                                violations.append(f"{py}:{side.lineno}: raw unit arithmetic")

            # Check format specs
            if isinstance(node, ast.JoinedStr):
                for v in node.values:
                    if isinstance(v, ast.FormattedValue) and v.format_spec:
                        spec = ast.unparse(v.format_spec)
                        if any(bf in spec for bf in BANNED_FORMAT_SPECS):
                            if _is_money_expr(v.value):
                                violations.append(f"{py}:{node.lineno}: banned format spec on money ident")

            # Check float() on _dollars / _fp
            if isinstance(node, ast.Call) and isinstance(node.func, ast.Name):
                if node.func.id == "float" and node.args:
                    arg = node.args[0]
                    if isinstance(arg, ast.Name) and (arg.id.endswith("_dollars") or arg.id.endswith("_fp")):
                        violations.append(f"{py}:{node.lineno}: float() on _dollars/_fp ident")

    assert not violations, "\n".join(violations)
```

```python
"""Shared FP migration helpers — scoped to debug logging of unknown API fields.

(Historical: also housed dollars_to_cents / fp_to_int converters; those
moved to talos.units as dollars_str_to_bps / fp_str_to_fp100 during the
bps/fp100 migration. This file now only hosts the per-session
unknown-fields logger.)
"""
from __future__ import annotations

from typing import Any

import structlog

logger = structlog.get_logger()

_seen_unknown: dict[str, set[str]] = {}  # model_name -> {field_names}


def log_unknown_fields(model_name: str, data: dict[str, Any], known: set[str]) -> None:
    """Log fields in *data* not in *known*, once per field per session."""
    unknown = data.keys() - known
    if not unknown:
        return
    seen = _seen_unknown.get(model_name)
    if seen is None:
        seen = _seen_unknown[model_name] = set()
    new = unknown - seen
    if not new:
        return
    seen.update(new)
    logger.debug("unknown_api_fields", model=model_name, fields=sorted(new))
```
